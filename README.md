# Simple Todo App

A simple and interactive to-do list application that allows users to manage daily tasks efficiently. The project focuses on refactoring existing JavaScript into an object-oriented structure using ES6 classes and modules.

## Functionality

The application provides the following features:

- A list of predefined to-do items generated using JavaScript
- Ability to add new to-do items through a popup form
- Form validation to ensure correct user input
- Each to-do item can be marked as completed
- To-do items can be deleted individually
- Unique IDs are generated for each to-do using UUID
- Forms reset correctly after successful submission

The project emphasizes clean architecture, modular code organization, and separation of concerns using Object-Oriented Programming principles.

## Technology

This project was built using:

- **HTML5** for structure
- **CSS3** for styling
- **JavaScript (ES6+)** for logic
- **JavaScript Modules** for code organization
- **Object-Oriented Programming (OOP)** with classes
- **UUID (via CDN)** for generating unique IDs
- **Live Server** for local development
- **Git & GitHub** for version control and deployment
